# MySQL

#### mysql 执行流程

1   mysql服务器做权限判断

2   判断mysql查询缓存是否开启
    
* 关闭
* 开启(从查询缓存中查询改语句有没有查询过)  

3   解析器对sql语法解析成一个解析树

4   预处理器对这棵树进行预处理得到优化后得解析树

5   查询优化器 根据解析树来判断这么执行 sql  得到 查询执行计划

6   查询执行计划 传递给 查询执行引擎

7   执行引擎 查询磁盘  io  

8   返回查询结果，如果开启缓存会把结果放一份到缓存中去


简：

*   权限判断
*   查询缓存是否开启
*   解析器解析
*   预处理器
*   查询优化器
*   查询执行计划
*   执行引擎
*   返回结果
